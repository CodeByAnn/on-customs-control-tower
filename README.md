# Customs Control Tower

D3.js data storytelling project inspired by On growth data and customs transformation work.

## Concept

On is scaling fast across regions, channels and product categories. The project connects public Q1 2026 business data with a simulated customs operations layer to show how growth, tariffs, product classification and shipment exceptions could become a digital control-tower workflow.

## Stack

- HTML
- CSS
- JavaScript
- D3.js via CDN
- Static, GitHub

## Data

Public business data is based on On Q1 2026 figures. Customs workflow and operational exception data are simulated for portfolio demonstration.
